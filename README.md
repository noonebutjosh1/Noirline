# Animated Dark-Themed Website 

A modern, responsive, multi-page website built with a strong focus on **UI design, smooth animations, and user experience**.  
This project showcases my ability to design and develop visually engaging websites using clean frontend technologies.

---

##  Project Overview

This website represents a **premium digital studio** concept, featuring:
- A dark, modern aesthetic
- Subtle motion and scroll-based animations
- Fully responsive layouts across all devices
- Clean and maintainable code structure

The goal of this project was to demonstrate **frontend design skills**, **animation handling**, and **responsive web development** suitable for real client work.

---

##  Pages Included

- **Home** – Hero section, animated elements, introduction
- **About** – Brand story and values
- **Services** – Clear breakdown of offerings
- **Contact / CTA sections** – Conversion-focused layout

---

##  Design & UX Features

- Dark UI with consistent color system
- Modern typography (Manrope & Space Grotesk)
- Glassmorphism-style cards
- Smooth hover effects and transitions
- Scroll reveal animations using `IntersectionObserver`
- Mobile-first responsive design

---

##  Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Flexbox, Grid, animations, media queries
- **JavaScript (Vanilla)** – Interactivity & scroll animations
- **IntersectionObserver API** – Performance-friendly animations

No frameworks or libraries were used — everything was built from scratch.

---

##  Responsiveness

The website is fully responsive and optimized for:
- Desktop
- Tablet
- Mobile devices

Navigation adapts seamlessly across screen sizes.
